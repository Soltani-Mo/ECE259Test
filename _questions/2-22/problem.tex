\problem{\textit{Parallel-plate capacitor with a dielectric in four parts}. A parallel-plate capacitor is filled with a dielectric composed of four parts, of permittivities $\varepsilon_1$, $\varepsilon_2$, $\varepsilon_3$ and $\varepsilon_4$, as in Fig.~Q2.9. Assuming that the capacitor is charged, that the electric field in each of the pieces is uniform, and that no surface free charges exist on dielectric-dielectric boundaries, consider the following four statements:
\begin{enumerate}[label=(\alph*)]\setlength{\parskip}{1pt}
\item if $\varepsilon_1 = \varepsilon_2$ and $\varepsilon_3 = \varepsilon_4$, then vector $\vec{E}$ is the same in all the pieces;
\item if $\varepsilon_1 = \varepsilon_2$ and $\varepsilon_3 = \varepsilon_4$, the vector $\vec{D}$ is the same in all the pieces;
\item if $\varepsilon_1 = \varepsilon_3$ and $\varepsilon_2 = \varepsilon_4$, then vector $\vec{E}$ is the same in all pieces;
\item if $\varepsilon_1 = \varepsilon_3$ and $\varepsilon_2 = \varepsilon_4$, then vector $\vec{D}$ is the same in all pieces;
\end{enumerate}
Which of the above statements are true?
\begin{enumerate}[label=(\Alph*)]\setlength{\parskip}{1pt}
\item Statements (a) and (b).
\item Statements (c) and (d).
\item Statements (a) and (d).
\item {Statements (b) and (c).}
\item None of the statements.
\end{enumerate}
