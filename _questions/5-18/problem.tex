\problem{\textit{Air gap in a simple linear magnetic circuit.} A thin toroidal core, made of a ferromagnetic material of permeability $\mu$, has an air gap, as shown in Fig.Q5.7. There is a time-invariant current through the winding. The magnitude of the magnetic field intensity vector in the ferromagnetic with respect to the clockwise reference direction is $H$. The magnitude of the magnetic field intensity vector in the gap ($H_0$) with respect to the same reference direction is
\begin{center}
\includegraphics[width=0.8\textwidth]{\npath Q5-7.png}
\end{center}
\begin{enumerate}[label=(\Alph*)]\setlength{\parskip}{1pt}
\item $H_0 = H$.
\item $H_0 = 0$.
\item $H_0 = \mu_0 H$.
\item $H_0 = \mu_0 H / \mu$.
\item {$H_0 = \mu H/ \mu_0$.}
\end{enumerate}}
