\problem{\textit{Insertion of a metallic slab in a uniform electric field.} 
A uniform electric field, of intensity vector $\vect{E}_0$, is established in the air-filled space between two metallic electrodes, as portrayed in Fig.~Q.16(a). If an uncharged (thick) metallic slab is then inserted in this space, without touching the electrodes, which gives the structure in Fig.~Q1.16(b), the electric field intensity vector in region 3 (above the slab) in the new electrostatic state is
\begin{enumerate}[label=(\Alph*)]\setlength{\parskip}{1pt}
\item $\vect{E}_3$ = 0.
\item $\vect{E}_3 = - \vect{E}_0$. 
\item $\vect{E}_3 = \vect{E}_0 / 3$.
\item $\vect{E}_3 = 3 \vect{E}_0 / 2$.
\item {$\vect{E}_3 = \vect{E}_0$.}
\end{enumerate}
